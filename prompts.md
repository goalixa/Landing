# Goalixa Landing Repository

## Purpose
Static marketing landing page for Goalixa - the first impression for potential users.

## Tech Stack
- **Pure HTML5**
- **Vanilla CSS3** (no frameworks)
- **Font Awesome** for icons
- **Responsive design** (mobile-first)

## Features
- Hero section with value proposition
- Features showcase
- Call-to-action buttons
- Responsive navigation
- Gradient background styling

## Code Conventions
- Semantic HTML5 elements
- Mobile-first responsive design
- CSS variables for theming
- Clean, maintainable CSS
- Font Awesome CDN for icons

## File Structure
```
goalixa-landing/
├── index.html             # Main landing page
├── style.css              # All styles
├── Dockerfile
├── docker-compose.yml
└── nginx.conf             # Nginx configuration (optional)
```

## Deployment
- Deployed as static site via nginx container
- Serves on `goalixa.com` root domain
- Minimal resource requirements

## Design Elements
- Purple gradient theme (#667eea to #764ba2)
- Clean, modern typography
- Mobile-responsive layout
- Simple navigation
- Feature cards with icons
